# Changelog

## 0.1.0

Initial release of `@spotify-confidence/react` - React hooks and components for Confidence feature flags.

### Features

- `ConfidenceProvider` - React context provider for pre-resolved flag bundles
- `useFlag` - Hook for accessing flag values with automatic exposure logging
- `useFlagDetails` - Hook for accessing flag values with manual exposure control
- Full type validation against default values
- Dot notation support for accessing nested flag properties
